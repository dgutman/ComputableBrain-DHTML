dhtmlxtreegrid

TreeGrid Extension

The user can output nested data items using special edition of dhtmlxGrid - dhtmlxTreeGrid. Additional JS file that should be added to the project is dhtmlxtreegrid.js . Look dhtmlxGrid manual to get details about grid structure initialization and usage.
