dhtmlxtreegrid

dhtmlxTreeGrid API Methods

dhtmlXGridObject Object

addRow add new row to treeGrid

addRowAfter add new row to treeGrid, after some other row

addRowBefore add new row to treeGrid, before some other row

closeItem collapse row

collapseAll collapse all tree structure

collectTreeValues get all possible values in column

deleteChildItems delete all childs of row in question

enableTreeCellEdit enable/disable editor of tree cell ; enabled by default

enableTreeGridLines enable lines in treeGrid

expandAll expand all tree structure

filterTreeBy filter grid by mask

getAllSubItems get list of id of all nested rows

getChildItemIdByIndex get id of child item at specified position

getItemImage get image of tree col

getItemText get real caption of tree col

getLevel return level of treeGrid row

getOpenState return open/close state of row

getParentId return id of parent row

getSubItems return list of child row id, sparated by comma

hasChildren return children count

openItem expand row

setFiltrationLevel allows to define , which level of tree must be used for filtering

setImageSize set size of treegrid images

setItemCloseable enable/disable closing of row

setItemImage set image of tree col

setItemText set real caption of tree col

setOnOpenEndHandler set function called after tree node opened/closed

setOnOpenStartHandler set function called before tree node opened/closed

sortTreeRows sorts treegrid by specified column

eXcell_tree Object

getLabel sets text representation of cell ( setLabel doesn't triger math calculations as setValue do)

setLabel sets text representation of cell ( setLabel doesn't triger math calculations as setValue do)
